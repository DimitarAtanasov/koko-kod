# Коко Код Академия

Приказна платформа за учене на програмиране — Курс 6+ (Роби) и Курс 11+ (Луна, Бо и Мия).

## Файлове
- `index.html` — цялото приложение (HTML+CSS+JS в един файл)
- `manifest.json` — PWA манифест (име, икони, цветове)
- `service-worker.js` — офлайн кеширане
- `icon-192.png`, `icon-512.png` — икони за инсталация

Всичките 5 файла + `.nojekyll` трябва да са в **корена на репото** (не в подпапка), за да съвпадат относителните пътища.

## Публикуване през GitHub Pages
1. Settings → Pages → Source: "Deploy from a branch" → Branch: `main` / `(root)` → Save
2. След ~1 минута сайтът е на `https://<потребител>.github.io/<репо>/`

## Обновяване по-късно
При всяка промяна на `index.html`, `manifest.json` или иконите — **повиши номера** в `service-worker.js`:
```js
const CACHE_NAME = 'koko-kod-v2';  // v1 -> v2 -> v3 ...
```
Иначе устройства, които вече са инсталирали приложението, ще продължат да виждат старата кеширана версия.

## Прогрес на децата
Пази се локално на устройството през вградената памет на артефакта (`window.storage`). Не се синхронизира между устройства — това е следваща стъпка, ако платформата порасне (истински бекенд с акаунти).
